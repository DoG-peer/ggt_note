なんども同じ単語を検索するなんてバカのすることです。

一度調べた単語は自分の言葉で説明を書いて
次に調べるときはそれを思い出すべきです。

ということで、これを実現するのに適したツールを作ってみました。

【注意】
data.txtは捨てないように。
捨ててしまったら
data.txtという名のファイルを作れば問題ありません。

【使い方】
ggt_note.exeを実行。
小さいめの検索窓が現れるので、
そこに調べたい単語を入力し、右の検索ボタンを押す。

登録がまだな単語ならブラウザが開いてグーグル検索されます。
登録が済んでいたら、最新の登録内容が表示されます。

同じ単語を複数回登録していた場合、
"<"や">"ボタンを押すと他のものが表示できます。

登録する方法は
小さいほうの窓に登録したい単語を書いて、
大きいほうにそれの説明を書き、登録ボタンを押す。
'''で始まる行があると正しく登録されないので、注意してください。

【動作環境】
.NET Frameworkの最新版が入っている必要があると思います。
OSはWindows7(64bit)でしか動作確認を取っていません。

【発展】
data.txtを直接いじって登録内容を変更する方法

data.txtは

'''単語
説明
'''

の形式の集まりで書かれています。
(終わりの'''の後ろはスペースも何も含んではいけません。
説明中に'''からはじまる行を入れるのは止めてください。
)

この塊を取り除けば、その登録を削除することになり、
逆に追加すれば、新しく登録することとなります。
改行はどれだけ入れてもらっても、常識の範囲内なら問題ありません。
初めが古い登録で行が進むにつれ新しい登録という扱いになります。


(現在この形式に沿わない場合への配慮は十分でないので、十分注意のうえ変更してください。)
